# HOW TO USE AUDIO

```
<!DOCTYPE html>
<html>
<body>
    <audio controls>
        <source src=
"https://media.geeksforgeeks.org/wp-content/uploads/20230524142525/gfg_offline_classes_en.mp3"
                type="audio/mp3">
        <source src=
"https://media.geeksforgeeks.org/wp-content/uploads/20220913101124/audiosample.ogg"
                type="audio/mp3">
    </audio>
</body>
</html>

```
